**MMCSS TWEAKER** — утилита для Windows 11, которая настраивает MMCSS (Multimedia Class Scheduler Service), подбирая профили под конкретный сетевой и аудио стек. Программа автоматически определяет тип драйвера (NDIS / NetAdapterCx), применяет согласованные значения, для достижения минимальной задержки и демонстрирует ожидаемые приоритеты потоков. В ручном режиме доступна точная настройка категорий планирования и приоритетов, а при необходимости можно быстро вернуть стандартные значения.
